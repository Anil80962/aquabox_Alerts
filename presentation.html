Features

Everything You Need

A complete water monitoring dashboard packed into a 5-inch touchscreen device.

🚨

Real-Time Alerts

Color-coded alert cards (High/Medium/Low) with auto-refresh every 2 minutes. Unread and read sections with live counters.

🔊

TTS Announcements

Text-to-Speech in 6 languages with typing animation synced to audio. Auto-announces new unread alerts.

📶

WiFi Manager

Scan, select, and connect to WiFi networks. Auto-reconnects on drop. Red icon when disconnected.

🔉

Volume Control

Speaker icon with volume slider. Adjusts system audio in real-time via amixer.

🤖

AquaGPT Chat

AI water assistant with typing animation, voice answers, and built-in water knowledge base.

🌐

Multi-Language

English, Telugu, Hindi, Kannada, Tamil, Malayalam. Switch language from header dropdown instantly.

🔒

Admin & Login

Secure admin panel for API credentials. Session persistence with auto-login on reboot.

📡

Bluetooth Audio

Connect to BT speakers for announcements. Scan and pair from the device.

⚡

Auto Recovery

Systemd service auto-restarts on crash. WiFi auto-reconnects and refreshes API on recovery.

WiFi Manager

Smart Connectivity

Built-in WiFi management right from the alert screen. No need for a separate app or SSH access.

⚫ Live Status Monitoring

WiFi icon checks connection every 5 seconds. White = connected. Red with X = disconnected. Auto-reconnects using wpa_cli.

📶 Network Scanner

Uses nmcli to scan available networks. Shows signal strength bars, lock icons for secured networks, and current connection indicator.

⌨ On-Screen Keyboard

wvkbd virtual keyboard for password entry on the touchscreen. Opens automatically when password field is shown.

🔄 Auto API Refresh

When WiFi reconnects after a drop, the system automatically refreshes API alerts — no manual intervention needed.

How It Works

From Boot to Alert

1

Boot

Pi boots, systemd starts aquabox-alerts service

2

Login

Auto-login with saved session or enter credentials

3

Fetch

API call to AquaGen for today's alerts

4

Display

Color-coded cards with unread/read sections

5

Announce

Auto TTS for new unread alerts in selected language

6

Refresh

Repeats every 2 min. WiFi auto-recovery built in.
